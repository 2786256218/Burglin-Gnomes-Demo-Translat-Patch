# Burglin-Gnomes-Demo-Translat-Patch  

- 我们对Burglin' Gnomes Demo游戏进行了汉化。我们解包并反编译了该游戏获取到了部分源代码，从根本上进行了翻译，并制作了一件翻译补丁install.bat。 

=== Burglin-Gnomes-Demo-Translat-Patch的优点：  

- 1.快捷方便一键安装  

- 2.可灵活修改添加自己的翻译内容在翻译字典  

- 3.持续更新  

- 4.安全可靠源代码公开透明  

 
# 构建 

- 如果你需要自行构建项目，我们也为你准备好了一件构建脚本build.bat，你只需要安装好所需要的SDK即可编译构建   

- 安装.Net Framework 4.0+（推荐）或MSBuild  

- 双击Build.bat即可  

 
# 使用  

- 首先你需要将Translate目录复制到游戏根目录下  

- 然后双击Install.bat即可完成汉化
